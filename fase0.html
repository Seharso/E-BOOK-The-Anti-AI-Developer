Fase 0: Fondasi & Mindset

Durasi: 1 Minggu

Tujuan: Menyiapkan mentalitas, alat, dan lingkungan kerja yang tepat. Ini adalah fondasi yang akan menentukan keberhasilan Anda.

Materi dalam Fase Ini

1. Mindset Pertumbuhan (Growth Mindset)

Penjelasan Detail

Pahami bahwa kemampuan teknis bisa diasah. "Saya belum bisa" bukan "Saya tidak bisa". Fokus pada proses, bukan hasil instan.

Proyek Nyata

Buat jurnal pembelajaran (bisa di Notion/Google Docs). Catat setiap hari apa yang dipelajari, kesulitan apa, dan bagaimana menyelesaikannya.

Tips & Trik

Celebrate small wins! Selesaikan satu sub-bab? Itu sebuah kemenangan.

Warning

Jangan bandingkan perjalanan Anda dengan developer senior yang sudah 10 tahun karirnya. Fokus pada diri sendiri.

Best Practice

Konsistensi > Intensitas. 1 jam setiap hari lebih baik daripada 7 jam sehari sekali seminggu.

Prompt AI Siap Pakai
[PERAN AI] Act as a motivational coach for a self-taught programmer. [TUJUAN] I'm feeling overwhelmed and think I'm not smart enough for this. [INSTRUKSI] Give me a 5-point pep talk to overcome imposter syndrome and reinforce the importance of a growth mindset. Use an encouraging and firm tone.

2. Menyiapkan Lingkungan Kerja

Penjelasan Detail

Install Visual Studio Code (VS Code) sebagai text editor. Install Git untuk version control. Buat akun GitHub. Install Node.js (nanti akan dipakai).

Proyek Nyata

-

Tips & Trik

Pelajari shortcut dasar VS Code (Ctrl+S, Ctrl+Z, Ctrl+Shift+P). Install ekstensi: Prettier, Live Server, GitLens.

Warning

Jangan install terlalu banyak ekstensi di awal, akan membingungkan.

Best Practice

Buat folder khusus untuk semua proyek pembelajaran Anda, misalnya C:\projects atau ~/projects.

Prompt AI Siap Pakai
[PERAN AI] Act as a DevOps expert. [TUJUAN] I'm setting up my development environment for the first time. [INSTRUKSI] Explain in simple terms what Git, GitHub, and VS Code are, why they are essential for a developer, and how they work together. Provide a step-by-step checklist for a complete beginner.

3. Konsep Dasar Git & GitHub

Penjelasan Detail

Pelajari perintah dasar: git init, git add, git commit, git push, git pull. Pahami konsep repository, branch, dan merge.

Proyek Nyata

Buat repository baru di GitHub. Clone ke komputer lokal. Buat file README.md. Lakukan add, commit, dan push.

Tips & Trik

Gunakan pesan commit yang jelas. Contoh: feat: add login button lebih baik dari update file.

Warning

Jangan pernah commit data sensitif (password, API key) ke GitHub.

Best Practice

Gunakan GitHub Pages untuk hosting proyek statis Anda nanti (gratis!).

Prompt AI Siap Pakai
[PERAN AI] Act as a senior developer mentoring a junior. [TUJUAN] I'm confused about git commit vs git push. [INSTRUKSI] Explain the difference using an analogy of writing and sending a letter. Also, provide 5 examples of good commit messages for a to-do list application project.

Proyek Nyata

Buat jurnal pembelajaran (bisa di Notion/Google Docs). Catat setiap hari apa yang dipelajari, kesulitan apa, dan bagaimana menyelesaikannya.

Best Practices & Tips

Best Practices untuk Fase 0:

  • Konsistensi lebih penting daripada intensitas - tetapkan waktu belajar harian
  • Gunakan journaling untuk melacak kemajuan dan refleksi
  • Siapkan lingkungan kerja dengan benar sejak awal
  • Pelajari shortcut editor untuk meningkatkan efisiensi
  • Jangan menyalin kode tanpa memahami fungsinya
